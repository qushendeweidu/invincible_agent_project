# 网络搜索技能

当此技能被触发时，supervisor 应将任务委派给 `search_agent`。

## 搜索策略指导

### 关键词优化
- query 必须简洁精准，控制在 3-8 个词
- 搜索时事新闻时，关键词中包含当前年份
- 优先使用中文关键词，技术类问题可用英文

### 结果处理
- 提取最相关的 3-5 条信息进行总结
- 对搜索结果进行交叉验证，避免单一来源
- 如需深入了解某条结果，可使用 crawl_webpage 抓取详情

### 回答格式
- 先给出直接答案，再补充来源和细节
- 注明信息的时效性（如"截至2026年..."）
- 如果搜索结果不确定，明确告知用户

## 脚本工具

### scripts/search_formatter.py — 搜索结果格式化器
**执行时机**: search_agent 获取到 bing_search 原始结果后，结果数量 > 5 条时调用
**功能**: 去重、按相关性排序、截断摘要、提取 top-N 结果
**调用方式**:
```bash
echo '{"results": [...], "query": "关键词"}' | python -m scripts.search_formatter --stdin --top 5
```
**输出**: JSON 格式的结构化搜索结果，包含 rank、title、url、snippet、domain
